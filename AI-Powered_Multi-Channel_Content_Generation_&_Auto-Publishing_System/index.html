AI-Powered Multi-Channel Content Generation & Auto-Publishing System

Role: Automation Specialist & AI Workflow Developer

Project Description

This project demonstrates an AI-powered automation system designed to generate and publish content across multiple social media platforms — Telegram, Facebook, LinkedIn, and Instagram.

The process starts when a user submits a YouTube video link and title. The system then automatically handles transcription, AI content generation, image creation, review, and auto-publishing — ensuring speed, consistency, and quality.

How It Works

  1. Input Form Submission: User submits a YouTube video link and title via an Airtable form.
  2. Trigger in Make.com: Submission activates an automated Make.com scenario.
  3. Video Transcription: DumplingAI transcribes the video into text.
  4. AI Content Creation: OpenAI generates unique text content for each social platform.
  5. Image Generation: Replicate creates a matching visual image for the post.
  6. Review & Approval: Generated content is reviewed before publishing.
  7. Auto-Publishing: Upon approval, the system posts automatically to all selected channels.

Tools Used

  • Airtable – database and form input
  • Make.com – automation and integrations
  • DumplingAI – YouTube transcription
  • OpenAI API – AI-driven content generation
  • Replicate – AI image generation

Benefits Delivered

  • End-to-end automation from video input to publication
  • Significant time savings and consistent quality output
  • Cross-platform content generation with unified workflow
  • Manual review step ensures quality control

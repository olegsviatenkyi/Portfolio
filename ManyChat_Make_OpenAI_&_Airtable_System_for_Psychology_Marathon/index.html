ManyChat, Make, OpenAI & Airtable System for Psychology Marathon

Role: Automation Specialist & Chatbot Developer

Overview

This project was built for a psychologist specializing in self-esteem, confidence, emotional balance, and overcoming fears. She also helps entrepreneurs grow sustainably while preventing burnout.

The goal was to create a smart, automated system to manage her online psychology marathon, covering user onboarding, AI-powered homework checking, and student support — all integrated into Telegram.

System Features

  • User Data Collection: via an Airtable form linked to Telegram.
  • Telegram Authorization: seamless participant onboarding through ManyChat.
  • CRM / EdTech Platform:
    • User database
    • Homework tracking
    • Task management
    • Automated homework check via AI
    • Task issuance by code words
    • Submission time tracking and student analytics
  • Notifications: instant curator alerts upon homework submission.
  • AI-Powered Communication:
    • Contextual replies and FAQs
    • Support for custom questions from students

How It Works

  1. A participant starts in the Telegram bot (built with ManyChat).
  2. They receive a link to an Airtable form and fill in their personal info.
  3. After submission, they get a code word unlocking their first task.
  4. Once they send the completed homework in chat — AI (OpenAI) reviews it and generates feedback.
  5. The feedback is sent to a curator for final review.
  6. The participant then receives the curator’s feedback + a new code word for the next task.
  7. The cycle continues until the marathon is complete.
  8. Throughout the process, the AI assistant also answers general and off-topic student questions.

Tools & Technologies

  • ManyChat – Telegram chatbot platform
  • Make.com – automation & integrations hub
  • OpenAI API – AI-powered homework analysis and reply generation
  • Airtable – database for users, tasks, and progress tracking

Result

The client received a fully automated, scalable marathon system where students move smoothly through lessons and receive real-time AI + curator feedback.

It significantly reduced manual work, improved student engagement, and provided clear analytics on progress and performance.

üìå Project Overview

This project demonstrates a fully automated two-way contact synchronization system built with Make.com, connecting HubSpot CRM and Google Sheets.

The system ensures that contact data stays consistent across both platforms without manual updates, even when records already exist on one or both sides.

The automation handles contact creation, updates, data merging, and missing field generation using clear priority rules and transformation logic.

‚ùó Problem

The client managed contacts in both HubSpot CRM and Google Sheets, which caused several issues:

  • Duplicate or outdated contact information
  • Manual copying between systems
  • Missing names for contacts created only from email addresses
  • No clear rule which system should override the other
  • High risk of data inconsistency over time

A reliable synchronization mechanism was needed to keep both systems aligned automatically.

‚úÖ Solution

I built a Make.com scenario that synchronizes contacts both ways between HubSpot and Google Sheets.

Key logic implemented:

  • Search for existing contacts in both systems
  • Create contacts or rows only when missing
  • Compare fields one by one before updating
  • Apply Google Sheets as the priority sourcer
  • Generate missing first and last names from email addresses
  • Update only empty or outdated fields

This approach avoids overwriting valid data and prevents sync loops.
